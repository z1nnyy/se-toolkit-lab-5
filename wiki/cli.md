## What is a CLI

A CLI (Command Line Interface) is a text-based interface for interacting with a computer program or [operating system](./operating-system.md#what-is-an-operating-system) by typing commands and reading text output.

Unlike a graphical user interface (GUI) that uses visual elements like buttons and menus, a CLI accepts typed commands and displays results as text.

In this course, you interact with a CLI using the [`VS Code Terminal`](./vs-code.md#vs-code-terminal), which runs a [shell](./shell.md#what-is-a-shell) such as [`bash`](./shell.md#bash).
